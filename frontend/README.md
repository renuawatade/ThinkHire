# Frontend (Vite + React + Tailwind)

This folder contains a minimal React frontend scaffold using Vite and Tailwind CSS.

Quick start:

1. From this folder run:

```bash
npm install
npm run dev
```

2. Open the dev server shown by Vite (usually http://localhost:5173).
